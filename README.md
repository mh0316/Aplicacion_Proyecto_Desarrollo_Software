# Aplicación de Denuncias de Faltas de Tránsito
Esta aplicación se complementa con el Proyecto de Desarrollo de Software enfocado en la creación de un plataforma de gestión de denuncias de tránsito disponible para toda la cuidadanía.

Es una aplicación de uso muy intuitivo para que cualquier persona pueda usarla, y que consta de las siguientes partes:
- Ventana de Inicio: Esta es la primera ventana que verá el usuario al abrir la aplicación, la cual cuenta con una sección de "Últimas Secciones" que muestra la última denuncia que hizo el usuario, además de datos estadísticos sobre la cantidad de denuncias en el último mes en general.
- Vetnana de Nueva Denuncia: En esta ventana el usuario podrá ingresar los datos referentes a la denuncia que quiera hacer seleccionando entre 7 tipos de faltas predefinidas, agregar una descripción sobre el hecho o la falta que presenció, luego deberá subir una evidencia visual de la falta que podrá ser en formato de foto o video, y deberá seleccionar la ubicación de la falta en un pequeño mapa integrado en la aplicación. Luego de haber subido los datos de estas 4 secciones, podrá clickear un botón en la parte inferior derecha de la ventana para revisar los datos de su denuncia y asegurarse de que todo esté correcto o, si el usuario lo requiere, volver a editar algún dato. Finalmente, luego de que el usuario haya confirmado los datos de su denuncia podrá, desde la misma sección donde hace la revisión de ésta, podrá seleccionar un botón para enviar la denuncia a un administrador de la plataforma donde será gestionada.
- Ventana de Ayuda: Esta es un sección que no involucra interacción y contiene una breve resumen de cómo hacer una buena denuncia.
